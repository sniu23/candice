
## 模板

- [egg-boilerplate-simple](https://github.com/eggjs/egg-boilerplate-simple)
- [element-starter](https://github.com/ElementUI/element-starter)

## 使用

```bash
# 前端开发 (使用模拟数据)
# open http://localhost:7001/
npm run front
# 前端打包（至 app/public）
npm run build
# 后端开发
# open http://localhost:7001/
npm run dev
# 生产部署 （后端 + 前端静态部署）
npm run start
```
